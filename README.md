Descripción
-----------
Este proyecto es una demostración de mis habilidades en HTML, CSS y Javascript, orientado a la creación de sitios web profesionales, accesibles y bien estructurados.
El objetivo es mostrar buenas practicas de desarrollo frontend y servir como base para futuros proyectos.

Caracteristicas
* Diseño responsivo adaptable a distintos dispositivos.
* Páginas principales: index.html, about.html, projects.html.
* Organización clara en la carpeta docs/ para despliegue en GitHub Pages.
* Código limpio y comentado para facilitar mantenimiento.
  
Archivos importantes
-------------------
- `index.html` - Página principal.
- `css/style.css` - Estilos principales (variables de color al inicio).
- `js/main.js` - Lógica de interactividad (nav móvil, modales, scroll).
- `images/` - Imágenes usadas (hero, avatar, servicios).
